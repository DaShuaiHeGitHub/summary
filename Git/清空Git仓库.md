# 清空 Git 仓库

##### 克隆仓库

```bash
git clone 仓库地址
```

##### 查询要回滚的 commit_id

```bash
git log
```

##### 查找并复制最早一次提交到的 commit_id

##### 强制还原到指定的 commit_id 版本中

```bash
git reset --hard commit_id
```

##### 删除仓库目录下所有文件和文件夹

##### 执行提交新 commit 前的常规操作

```bash
git add .
git commit -m "提交信息"；
```
